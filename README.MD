## ✨ Features

* ⚡ No CSS files needed
* 🧠 IntelliSense (auto suggestions)
* 🎯 Faster development
* 🪶 Lightweight & zero config
* 🔥 Works with HTML & React

---

## 🚀 Quick Example

```html
<div class="p4 bg-blue-5 t-white">
  Hello Uvwind 🚀
</div>
```

---

## 🧠 How it works

Uvwind dynamically maps utility classes to styles —
no CSS file, no build step, no config.
